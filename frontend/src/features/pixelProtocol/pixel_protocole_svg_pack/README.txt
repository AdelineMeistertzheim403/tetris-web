# Pixel Protocole SVG Pack

Contenu :
- 100 décorations SVG optimisées pour Phaser/Canvas
- 40 décorations SVG "3D" isométriques supplémentaires
- 3 backgrounds parallax SVG
- 1 tileset cyber complet + JSON de mapping

Conseils :
- Les SVG n'utilisent pas de filtres lourds pour rester compatibles Canvas/Phaser.
- Utilise `shape-rendering="crispEdges"` pour garder un rendu pixel net.
- Le fichier JSON du tileset donne les coordonnées de chaque tuile dans l'atlas SVG.
